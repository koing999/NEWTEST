# 💼 투자 경제성 분석 대시보드 - 사용법 가이드

## 🚀 빠른 시작 (30초 완성)

### 1단계: 기본 설정
1. **매출 자동 생성** 클릭
   - 초기 매출: 100억원 (기본값)
   - CAGR: 15% (기본값)
   
2. **"🚀 분석 시작"** 버튼 클릭

### 2단계: 결과 확인
- **NPV**: 순현재가치 (양수면 투자 가능)
- **IRR**: 내부수익률 (WACC보다 높으면 OK)
- **회수기간**: 투자금 회수까지 걸리는 시간
- **손익분기**: 누적 영업이익이 투자금 회수하는 시점

## 📊 탭별 상세 사용법

### 📊 기본 설정 탭

#### 1) 매출 데이터 입력 (2가지 방법)

**방법 A: CAGR 자동 생성** ⭐ 추천
```
초기 매출: 100억원
CAGR: 15%
→ "매출 자동 생성" 클릭
```

**방법 B: 엑셀 업로드**
```
파일 형식:
연도 | 매출(억원)
1    | 100
2    | 120
3    | 145
```

#### 2) 기본 가정 입력
- **초기 투자금**: 500억원 (설비, 개발비 등)
- **내용연수**: 10년 (감가상각 기간)
- **분석 기간**: 10년 (1~20년)
- **법인세율**: 25% (한국 기본 세율)
- **운전자본 비율**: 10% (매출의 몇 %를 운전자금으로 묶을지)

---

### 🧮 WACC 탭

#### 1) 산업별 빠른 적용 ⚡
```
IT/소프트웨어 → WACC ~8%
제조업 → WACC ~10%
유통/소매 → WACC ~9%
헬스케어 → WACC ~7%
건설 → WACC ~11%
```

#### 2) WACC 직접 계산 (고급)

**Step 1: Beta 조정**
```
Unlevered β: 0.8 (동종업계 평균)
목표 D/E: 40% (부채/자기자본 비율)
→ "Levered β 계산" 클릭
```

**Step 2: CAPM 계산**
```
무위험이자율: 3.5% (국고채 금리)
시장수익률: 9% (KOSPI 역사적 평균)
Beta: 1.2 (Step 1에서 계산됨)
→ "자기자본비용 계산" 클릭
```

**Step 3: WACC 계산**
```
부채비중 D/(D+E): 30%
타인자본비용: 5% (대출 이자율)
자기자본비용: 12% (Step 2에서 계산됨)
→ "WACC 계산" 클릭
```

---

### 🎭 시나리오 탭

#### 1) 잔존가치 (Terminal Value) 선택

**청산가치** (보수적)
```
마지막 해에 자산 처분 가격
예: 100억원
```

**영구성장** (일반적) ⭐ 추천
```
영구성장률: 2% (GDP 성장률 수준)
공식: TV = FCF × (1+g) / (WACC-g)
```

**Exit Multiple** (M&A 상황)
```
Exit Multiple: 10배
기준 지표: EBITDA 또는 매출
예: 마지막 해 EBITDA × 10배
```

#### 2) 시나리오 선택
- **보수적**: 매출 80%, WACC 12%
- **기본**: 현재 입력값 그대로
- **낙관적**: 매출 120%, WACC 8%

## 📈 결과 분석 방법

### 1) KPI 카드 해석

| 지표 | 의미 | 판단 기준 |
|------|------|-----------|
| NPV | 투자로 얻는 순이익 | > 0 이면 투자 가능 |
| IRR | 내부수익률 | > WACC 면 투자 가능 |
| 회수기간 | 투자금 회수 시간 | 짧을수록 좋음 (보통 5년 이내) |
| 손익분기 | 본전 뽑는 시점 | 빠를수록 좋음 |

### 2) 몬테카를로 시뮬레이션

**실행**: "시뮬레이션 실행" 클릭 (1000회)

**결과 해석**:
- **P10**: 최악의 경우 (10% 확률로 이보다 나쁨)
- **P50**: 중앙값 (50% 확률)
- **P90**: 최선의 경우 (10% 확률로 이보다 좋음)

**위험도 판단**:
```
P10이 양수 → 위험 낮음 ✅
P10이 음수, P50 양수 → 중간 위험 ⚠️
P50이 음수 → 위험 높음 ❌
```

### 3) 민감도 분석

**1-Way 민감도**:
- 매출, WACC, 운전자본 각각 변할 때 NPV 변화
- **가장 민감한 변수** 확인 가능

**2-Way 민감도** (매출 × WACC):
- 두 변수 동시 변화 시나리오
- 녹색 = NPV > 0, 빨간색 = NPV < 0

### 시나리오 1: 제조업 설비 투자

```
📊 기본 설정:
- 초기 투자금: 1,000억원
- 초기 매출: 200억원
- CAGR: 10%
- 내용연수: 15년

🧮 WACC:
- 산업 선택: 제조업 (10%)

🎭 시나리오:
- 잔존가치: 영구성장 (2%)

✅ 결과:
- NPV: 150억원 → 투자 가능
- IRR: 12.5% → WACC 10%보다 높음 ✅
```

### 시나리오 2: IT 스타트업 투자

```
📊 기본 설정:
- 초기 투자금: 50억원
- 초기 매출: 10억원
- CAGR: 40% (고성장)
- 내용연수: 5년

🧮 WACC:
- 산업 선택: IT (8%)

🎭 시나리오:
- 잔존가치: Exit Multiple (15배, 매출 기준)
- 시나리오: 낙관적 선택

✅ 결과:
- NPV: 500억원 → 높은 수익성 ✅
- IRR: 65% → 초고수익 ✅
```

### 시나리오 3: 보수적 유통업 투자

```
📊 기본 설정:
- 초기 투자금: 300억원
- 초기 매출: 150억원
- CAGR: 5% (저성장)
- 내용연수: 10년

🧮 WACC:
- 산업 선택: 유통/소매 (9%)

🎭 시나리오:
- 잔존가치: 청산가치 (50억원)
- 시나리오: 보수적 선택

- NPV: -20억원 → 투자 불가 ❌
- IRR: 8.5% → WACC 9%보다 낮음 ❌
```

## ⚠️ 주의사항

### 1) 입력값 검증
- **음수 입력 불가**: 매출, 투자금, 세율 등
- **WACC > 영구성장률**: 아니면 계산 오류
- **분석 기간**: 1~20년 범위

### 2) 계산 가정
- **영업이익률**: 기본 20% (고정)
- **감가상각**: 정액법 (내용연수로 균등 배분)
- **NWC 회수**: 마지막 해에 전액 회수

### 3) 브라우저 호환성
- **권장**: Chrome, Edge, Safari 최신 버전
- **비권장**: IE (지원 안 함)

## 🆘 자주 묻는 질문 (FAQ)

### Q1. IRR이 음수로 나와요!
**A**: 투자금 대비 현금흐름이 부족한 경우입니다.
- 매출 증가율 높이기
- 초기 투자금 줄이기
- 영업이익률 개선

### Q2. 손익분기가 10년(끝)으로 나와요!
**A**: 누적 영업이익이 투자금을 회수하지 못한 경우입니다.
- 투자 불가 신호 ⚠️
- 사업 모델 재검토 필요

### Q3. 2-Way 민감도 분석이 전부 빨간색이에요!
**A**: 대부분의 시나리오에서 NPV가 음수입니다.
- 투자 매력도 낮음
- 매출 또는 WACC 전제 재검토

### Q4. 몬테카를로 P50이 음수인데 NPV는 양수예요!
**A**: 몬테카를로는 랜덤 변동을 줍니다.
- 기본 NPV: 확정적 가정
- 몬테카를로: 불확실성 반영 (더 보수적)

### Q5. 차트가 안 보여요!
**A**: 브라우저 캐시 문제일 수 있습니다.
- Ctrl+Shift+R (강력 새로고침)
- 시크릿 모드로 실행

## 🎓 용어 사전

| 용어 | 뜻 |
|------|-----|
| NPV | Net Present Value (순현재가치) |
| IRR | Internal Rate of Return (내부수익률) |
| WACC | Weighted Average Cost of Capital (가중평균자본비용) |
| FCF | Free Cash Flow (잉여현금흐름) |
| NOPAT | Net Operating Profit After Tax (세후순영업이익) |
| CAPM | Capital Asset Pricing Model (자본자산가격결정모형) |
| Beta | 시장 위험 대비 개별 자산의 위험도 |
| Terminal Value | 분석 기간 이후의 잔존 가치 |
| NWC | Net Working Capital (순운전자본) |
